CS2 Float Scanner

Privacy Policy — Effective Date: March 25, 2026

1. Overview

CS2 Float Scanner is a Google Chrome browser extension that displays Float Value, Paint Seed, sticker details, keychain patterns, rare pattern tiers, price chart analysis and alerts for Counter-Strike 2 items on Steam Community Market pages and Steam inventories.

Core principle: the extension operates almost entirely locally. It does not collect, store, or transmit any personal information of its users. No accounts or sign-ups are required.

2. Data Processed by the Extension

DataSourcePurposeShared with third parties
Float Value, Paint Seed, sticker/keychain info Steam API (steamcommunity.com), already loaded on the page Display next to each item No
User settings User input (float precision, feature toggles, theme) Personalize extension behavior No
Price/float tiers, custom pattern tiers User input Highlight matching items on the market No
Notification thresholds User input Send local browser notifications No
Milestone state (shown/not shown) Generated locally Show one-time achievement notifications at user count milestones No

3. Data Storage

All data is stored exclusively in the user's browser via chrome.storage.local. This includes:

  • Display settings (float precision, feature toggles, theme)
  • Custom price/float tiers and pattern tiers
  • Notification thresholds
  • Milestone achievement state
  • Language preference

Data never leaves the user's device.

4. Extension Permissions

PermissionWhy it is needed

5. Network Requests

The extension makes the following network requests:

a) Steam Community (steamcommunity.com)

Requests to fetch listing data during bulk loading (20/100 items) and price history for chart analysis. These requests use the user's existing Steam session cookies and are directed exclusively to Valve's servers.

b) Shields.io (img.shields.io)

External request: the extension makes a single request to img.shields.io no more than once per day to retrieve the current number of extension users from the Chrome Web Store. This is used solely to trigger one-time milestone achievement notifications (at 100, 500 and 1,000 users). Once the 1,000-user milestone is reached, no further requests are made.

This request:

  • Does not send any user data, cookies, or identifying information
  • Is a simple anonymous GET request to a public API
  • Returns only the total number of extension users (a single integer)
  • Can be disabled by reaching the final milestone; after that, no external requests are made

The extension does not communicate with any other external servers.

6. Analytics and Tracking

The extension does not use any analytics, tracking, telemetry, or advertising systems. No Google Analytics, Facebook Pixel, or any similar services are used.

7. Children

The extension is not intended for children under the age of 13 and does not knowingly collect data from such users.
