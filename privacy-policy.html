1. Общие сведения

CS2 Float Scanner — расширение для браузера Google Chrome, которое отображает Float Value и Paint Seed предметов Counter-Strike 2 на страницах Steam Community Market и в инвентарях Steam.

Главный принцип: расширение работает полностью локально. Никакие данные не отправляются на внешние серверы. Расширение не собирает, не хранит и не передаёт персональную информацию пользователей.

2. Какие данные обрабатывает расширение

ДанныеИсточникЦельПередача третьим лицам

Пользовательские настройки Выбор пользователя (точность float, переключатели функций) Персонализация работы расширения Нет
Градации цена/float Ввод пользователя Подсветка выгодных предложений Нет
Порог уведомлений Ввод пользователя Отправка локальных уведомлений браузера Нет

3. Хранение данных

Все данные хранятся исключительно локально в браузере пользователя через chrome.storage.local. Данные включают:

  • Настройки отображения (точность float, переключатели функций)
  • Пользовательские градации цена/float для конкретных предметов
  • Пороги уведомлений

Данные никогда не покидают устройство пользователя.

4. Разрешения расширения

РазрешениеЗачем необходимо

host_permissions для steamcommunity.com Доступ к данным на страницах Steam Market и инвентарей для извлечения информации о предметах

5. Сетевые запросы

Расширение выполняет запросы только к steamcommunity.com — для получения данных о листингах при массовой загрузке (20/100 предметов). Эти запросы используют существующие cookies пользователя Steam и направлены исключительно к серверам Valve. Расширение не взаимодействует ни с какими другими серверами.

6. Аналитика и отслеживание

Расширение не использует никакие системы аналитики, трекинга, телеметрии или рекламы. Не используются Google Analytics, Facebook Pixel или любые другие подобные сервисы.

7. Дети

Расширение не предназначено для детей младше 13 лет и не собирает намеренно данные от таких пользователей.

8. Изменения в политике

При внесении существенных изменений в данную политику обновлённая версия будет опубликована на этой же странице с указанием новой даты вступления в силу.

1. Overview

CS2 Float Scanner is a Google Chrome browser extension that displays the Float Value and Paint Seed of Counter-Strike 2 items on Steam Community Market pages and Steam inventories.

Core principle: the extension operates entirely locally. No data is sent to external servers. The extension does not collect, store, or transmit any personal information of its users.

2. Data Processed by the Extension

DataSourcePurposeShared with third parties
Float Value, Paint Seed Steam API (steamcommunity.com), already loaded on the page Display next to each item No
User settings User input (float precision, feature toggles) Personalize extension behavior No
Price/float tiers User input Highlight good deals No
Notification thresholds User input Send local browser notifications No

3. Data Storage

All data is stored exclusively in the user's browser via chrome.storage.local. This includes:

  • Display settings (float precision, feature toggles)
  • Custom price/float tiers for specific items
  • Notification thresholds

Data never leaves the user's device.

4. Extension Permissions

PermissionWhy it is needed
storage Save user settings and price/float tiers between sessions
notifications Send browser notifications when an item with float below the set threshold is found
host_permissions for steamcommunity.com Access data on Steam Market and inventory pages to extract item information

5. Network Requests

The extension only makes requests to steamcommunity.com — to fetch listing data during bulk loading (20/100 items). These requests use the user's existing Steam cookies and are directed exclusively to Valve's servers. The extension does not communicate with any other servers.

6. Analytics and Tracking

The extension does not use any analytics, tracking, telemetry, or advertising systems. No Google Analytics, Facebook Pixel, or any similar services are used.

7. Children

The extension is not intended for children under the age of 13 and does not knowingly collect data from such users.
