v1.8.0 April 2026 Latest

New

  • Price forecast — Holt-Winters smoothing model predicts item price one month ahead with optimistic, base and pessimistic scenarios
  • Sticker prices on market listings — shows price per sticker (Steam Market median, USD), total sticker value and overpay estimate for combo stickers
  • Deep Order recommendations — analyzes 30 days of sales history and current order book to suggest two buy order prices: aggressive (max profit) and moderate (more fills)
  • Deep Order trend correction — if price is rising, recommended prices are adjusted upward so orders stay realistic
  • Deep Order target quantity — set how many items you want to buy per month and prices recalculate to match that volume
  • Real screenshot card export via external site (Swap.gg integration in share modal)
  • Sticker prices sourced daily from ByMykel/counter-strike-price-tracker on GitHub

Improved

  • Chart and Deep Orders buttons now appear side by side near the price graph, also on sticker and case pages
  • Chart analysis button always renders; Deep Orders works independently of chart settings
  • Privacy policy updated to reflect sticker price data source and histogram requests
v1.7.0 March 2026

New

  • Enhanced market pagination with backward compatibility (Steam native, SIH), jump to last page, page number input
  • Advanced technical indicators: SMA 50, SMA 200, ATR (separate sub-chart), OBV (on volume chart)
  • Drawing tools on price chart — horizontal lines and trend lines, double-click to delete
  • Custom time periods for chart: 14D, 60D, 6M, 1.5Y, 2Y presets and custom days input
  • Milestone Easter eggs — Steam Achievement Unlock-style notifications at 100, 500, and 1,000 users
  • Extension landing page at cs2floatscanner.github.io

Fixed

  • Copying a float value no longer resets pagination state

Improved

  • Indicator toggles grouped by category with pill-style ON/OFF buttons
  • Chart area scrollable with custom-styled scrollbar
  • Milestone user count checked via shields.io API (no CORS issues)
v1.6.0 March 2026

New

  • Built-in rare patterns database with 40+ items: Case Hardened Blue Gems for AK-47, Five-SeveN, MAC-10, Karambit, M9 Bayonet, Butterfly Knife, and 15+ more knives; Heat Treated patterns for Desert Eagle and Five-SeveN; Color scale tiers for 20 charms (Die-cast AK, Lil' SAS, Hot Howl, Semi-Precious, etc.)
  • Full Charm/Keychain support on Steam Market — shows pattern number instead of float
  • Pattern-based filtering and sorting for charm items with purple accent badges
  • Pattern Tier ranges — define tiers like 20000-21500 for charms
  • Up to 10 custom pattern tiers with 200 entries each
  • In-page toast notifications with item details and "Go to listing" button
  • Alert matching listings highlighted with pulsing red outline and auto-scrolled into view
  • Audio alert (short dual-tone beep) when a float below threshold is found
  • Duplicate alert prevention per listing ID
  • Changelog page with "What's New" link in extension settings

Fixed

  • Float alerts now reliably trigger on page reload and page navigation
  • Fixed chrome.runtime.sendMessage promise rejection not being caught (MV3)
  • Fixed background service worker not calling sendResponse for notification messages
  • Built-in tier display now shows range strings instead of "0 patterns" for range-based tiers

Improved

  • Notifications work even when OS-level Chrome notifications are suppressed (toast fallback)
  • Build script now auto-deploys changelog to GitHub Pages via SSH
